gem5.prebuilt.viper.board module¶

class gem5.prebuilt.viper.board.ViperBoard(clk_freq: str, processor: AbstractProcessor, memory: AbstractMemorySystem, cache_hierarchy: AbstractCacheHierarchy, gpus: List[BaseViperGPU] | None = None)¶

Bases: X86Board

A derivative of X86Board capable of full system simulation for X86 with a GPU device. Provides all the functionality of the X86Board with helper methods specific to booting a disk with GPU libraries installed.
